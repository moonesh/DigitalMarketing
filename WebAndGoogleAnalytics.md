# Web and Google Analytics  

You can track following : using [https://analytics.google.com/analytics/web/#/](https://analytics.google.com/analytics/web/#/)
  - Behaviour  
  - Time Spent  
  - Which Pages  
  - Visitor Count  
  - Conversion  
  - Gender  
  - Age  
  - Location  
  - Device  
  - Browser
